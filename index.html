Analisis Getaran Struktur dan Prediksi Frekuensi Natural

Penentuan Perpindahan Struktur Multi-Titik dan Prediksi Frekuensi Natural

Menggunakan Metode Gauss-Jordan dan Analisis Regresi Linier

Metode Penyelesaian

Eliminasi Gauss-Jordan

Algoritma aljabar linier untuk mengubah matriks augmented menjadi bentuk eselon baris tereduksi (RREF). Metode ini memberikan solusi langsung dan eksak untuk sistem persamaan linier simultan (Ax=B).

Metode Pemodelan

Regresi Linier (Least Squares)

Pendekatan statistik untuk memodelkan hubungan antara variabel dependen (f²) dan independen (1/m) dengan meminimalkan jumlah kuadrat residu (selisih data eksperimen vs model).

Model Matematika

Sistem dimodelkan dalam bentuk matriks augmented \([A|B]\) dan diselesaikan untuk mencari vektor perpindahan \(\{x\}\).

x = Perpindahan (Variabel dicari)
F = Gaya Eksternal

regression_analysis.py Python / SciPy 
from scipy import stats
import numpy as np

# 1. Data Transformasi (f^2 vs 1/m)
m = np.array([5, 10, 15, 20, 25, 30, 35])
f = np.array([12.8, 9.2, 7.5, 6.4, 5.7, 5.2, 4.8])

X_trans = 1 / m       # Sumbu X
Y_trans = f ** 2      # Sumbu Y

# 2. Regresi Linier (Least Squares)
slope, intercept, r, p, err = stats.linregress(X_trans, Y_trans)

# 3. Hitung Parameter Fisik
# Rumus: slope = k / (4 * pi^2)
k = slope * 4 * (np.pi ** 2)

print(f"Kekakuan (k): {k:.2f} N/m")
# Output: 32610.74 N/m

Interpretasi Fisika: Dinamika Struktur

Efek Massa (\(f \propto 1/\sqrt{m}\))

Inverse Relation

Massa berbanding terbalik dengan frekuensi natural. Menambahkan massa 10x lipat tidak menurunkan frekuensi 10x lipat, tapi akar kuadratnya. Ini penting untuk tuning agar struktur tidak resonansi dengan mesin.

Validasi Hukum Hooke

Elastic Region k ≈ 32.6 kN/m

Linearitas tinggi (\(R^2 \approx 1\)) mengonfirmasi bahwa material bekerja di daerah elastis. Tidak ada indikasi *plastic deformation* atau kerusakan pegas selama eksperimen.

Kesimpulan Akhir

Berdasarkan analisis numerik, struktur menunjukkan perilaku elastis linier yang konsisten. Model regresi memberikan prediksi yang sangat akurat untuk beban dinamis. Namun, perhatian khusus harus diberikan pada Node 5 dan sambungan Node 3-4 karena menunjukkan anomali perpindahan statis yang signifikan.
