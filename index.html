Analisis Numerik Getaran Pegas

Penerapan Metode Regresi Linier untuk Estimasi Parameter Fisika

Metode: Least Squares Model: Harmonik Sederhana Tools: Python (NumPy/SciPy)

üìö Teori Metode Numerik

Analisis ini menggunakan metode Regresi Linier Kuadrat Terkecil (Least Squares Method) untuk meminimalkan error kuadrat antara data eksperimen dan model teoretis.

1. Linearisasi Model:

$$ f = \frac{1}{2\pi} \sqrt{\frac{k}{m}} \Rightarrow f^2 = \left(\frac{k}{4\pi^2}\right) \cdot \frac{1}{m} + C $$

Dimana $y = f^2$, $x = 1/m$, dan gradien $b = k/4\pi^2$.

2. Matriks Normal:

$$ \begin{bmatrix} n & \sum x \\ \sum x & \sum x^2 \end{bmatrix} \begin{bmatrix} a \\ b \end{bmatrix} = \begin{bmatrix} \sum y \\ \sum xy \end{bmatrix} $$

üíª Implementasi Kode

Cuplikan kode Python utama untuk komputasi regresi:

# Transformasi Data
x = 1 / m   # Sumbu X
y = f**2    # Sumbu Y

# Komputasi Statistik (SciPy)
slope, intercept, r, p, err = stats.linregress(x, y)

# Perhitungan Fisika (Konstanta Pegas)
# slope = k / (4 * pi^2)
k = slope * 4 * (np.pi**2)

print(f"Konstanta Pegas k: {k:.2f} N/m")
print(f"Akurasi Model (R^2): {r**2:.4f}")
